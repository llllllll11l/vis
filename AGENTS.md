# AGENTS.md

## Projects Overview
This is a pure frontend data visualization project built with Vue, JavaScript, Vite, and Apache ECharts. 
It loads local JSON/CSV files from public/data and renders interactive charts in the browser.
It shows changes in House Price and Population in recent 20 or 30 years, as well as the internal connection between them.

## Tech Stack
- Framework: Vue
- Language: JavaScript
- Build Tool: Vite
- Visualization: Apache ECharts
- Data Source: local JSON / CSV files
- CSV Parser: PapaParse
- Styling: CSS / SCSS
